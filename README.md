## 🛠 Instalación y desarrollo local

#### Clonar el repositorio
```bash
git clone https://github.com/Alexa2403-progm/website-portfolio.git
```

#### Entrar al proyecto
```
cd my-portfolio-dev
```

#### Instalar dependencias
```
npm install
```

#### Iniciar en modo desarrollo
```
npm run dev
```
